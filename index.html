Remote Control:

  • X Button (Y/T): Scroll Down
  • B Button (J/N): Scroll Up
  • Y Button (U/F): Increase Font Size
  • A Button (H/R): Decrease Font Size
  • Mirror Button (O/G): Play/Pause
  • Down Button (A/Q): Scroll Down
  • Up Button (D/C): Scroll Up

Keyboard Controls:

  • Space: Play/Pause
  • Up Arrow: Increase Speed
  • Down Arrow: Decrease Speed
  • Y: Increase Font Size
  • A: Decrease Font Size
  • O: Toggle Mirror
